Local Development (From Git Clone)

Clone outside your Laravel project:

https://github.com/brrittoo/field-dynamic-encryption.git


Add as a path repository:

"repositories": [
    {
        "type": "path",
        "url": "../field-dynamic-encryption",
        "options": { "symlink": true }
    }
]


Require locally:
composer require brrittoo/field-dynamic-encryption:@dev
Publish configuration:

php artisan vendor:publish --tag=field-encryption-config



âœ… Symlinks the clone; changes reflect immediately.
